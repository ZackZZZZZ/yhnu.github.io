PyCharm_2019.1.1_破解

Python 环境搭建

0x01 下载2019.1.1专业版本

https://www.jetbrains.com/pycharm/download/other.html

0x02 将jetbrains-agent.jar放置到Pycharm安装目录的bin目录下

0x03 修改xx.vmoptions 文件，加上 -javaagent:C:\Program Files\JetBrains\PyCharm 2019.1.1\bin\jetbrains-agent.jar

jetbrains-agent下载地址

0x04 激活

启动Pycharm，并激活，如图：选择最后一种License server激活方式，地址填入：http://jetbrains-license-server （应该会自动填上），或者点击按钮：”Discover Server”来自动填充地址，完成激活
